TELEGRAM API SETUP GUIDE

텔레그램 알림 설정: 봇 토큰 & 채팅 ID 발급 방법

사용자님의 알림 시스템(모니터링 앱)이 텔레그램으로 메시지를 보낼 수 있도록 봇 토큰(Bot Token)과 채팅 ID(Chat ID)를 발급받는 상세 가이드입니다. 이 두 정보는 알림 전송에 필수적입니다.

1. 🤖 봇 토큰 발급 (BotFather)

텔레그램 봇을 생성하고 고유의 API 토큰을 발급받는 과정입니다. BotFather라는 공식 봇을 이용합니다.

1 텔레그램에서 BotFather 검색

텔레그램 앱/PC에서 검색창에 @BotFather를 검색하여 대화를 시작합니다. (파란색 체크 표시가 있는 공식 봇입니다.)

2 봇 생성 명령어 입력

/newbot

3 봇 이름 및 사용자 이름 설정

  • 먼저, 사용자에게 보여줄 봇의 이름 (예: My Trading Alert)을 입력합니다.
  • 다음으로, 봇의 사용자 이름 (예: MyTradingAlert_bot)을 입력합니다. 사용자 이름은 반드시 _bot으로 끝나야 합니다.

4 봇 토큰 복사

봇 생성이 완료되면, BotFather가 메시지와 함께 긴 문자열을 보내줍니다. 이것이 바로 봇 토큰입니다. `Use this token to access the HTTP API:` 바로 아래에 있는 문자열을 복사하여 모니터링 앱의 Bot Token 입력란에 붙여넣습니다.

예시: 123456789:AAFg8S-QxY_tG9WfM0Oa1bC2dE3fG4hI5jK6lM7nO8pQ9rS

2. 💬 채팅 ID 확인 (ID Bot)

봇이 메시지를 보낼 대상을 지정하기 위한 고유 ID입니다. ID 확인 전용 봇을 이용합니다.

1 ID 확인용 봇 검색 및 메시지 전송

텔레그램에서 @get_id_bot 또는 @userinfobot을 검색하여 대화를 시작합니다.

봇에게 아무 메시지나 보내거나 (/start) 명령어를 입력합니다.

2 채팅 ID 복사

봇이 응답으로 보내주는 `ID: 123456789`와 같은 형태의 숫자 값을 복사하여 모니터링 앱의 Chat ID 입력란에 붙여넣습니다.

🔔 중요: 그룹 채팅방 ID를 사용하는 경우
알림을 그룹 채팅방으로 받고 싶다면, 발급받은 봇을 그 그룹에 초대해야 합니다. 그룹 채팅방의 ID는 일반적으로 마이너스(-) 기호로 시작합니다 (예: -100123456789). 반드시 이 마이너스 기호를 포함하여 입력해야 합니다.
