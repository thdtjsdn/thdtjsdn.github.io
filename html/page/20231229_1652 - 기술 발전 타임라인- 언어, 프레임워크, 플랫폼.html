Python 0.9
Language
1989년 12월

Python은 Guido van Rossum에 의해 개발된 고급 프로그래밍 언어로, 1989년 12월에 첫 버전인 Python 0.9.0이 공개되었습니다.

공식 출시는 1991년 2월에 공개된 Python 1.0입니다. 다양한 운영 체제에서 사용 가능하며 가독성이 높습니다.

간결하고 가독성이 높은 문법: 적은 양의 코드로 많은 작업을 수행할 수 있도록 설계되었습니다.

동적 타이핑 및 자동 메모리 관리: 변수의 타입을 선언할 필요가 없으며 자동으로 메모리를 관리합니다.

풍부한 표준 라이브러리: 다양한 작업을 간편하게 수행할 수 있는 라이브러리를 제공합니다.

높은 확장성: 웹 개발, 데이터 분석, 인공 지능 등 다양한 분야에서 활용됩니다.

R
Language
1993년 01월 29일

R 언어는 통계 분석 및 데이터 시각화를 위한 프로그래밍 언어로, 1993년에 Ross Ihaka와 Robert Gentleman에 의해 개발되었습니다.

통계 분석 및 데이터 시각화: 주로 통계 및 데이터 분석을 위해 설계되었으며 다양한 통계 기법과 시각화 도구가 내장되어 있습니다.

풍부한 패키지 생태계: 수많은 패키지를 포함한 풍부한 생태계를 가지고 있어 기능을 확장할 수 있습니다.

벡터화 및 함수형 프로그래밍 지원: 벡터화 연산을 지원하여 효율적인 데이터 연산을 수행하며, 함수형 프로그래밍 개념을 내장합니다.

데이터 프레임: 데이터를 표 형태로 저장하고 조작할 수 있는 데이터 프레임 구조를 제공합니다.
